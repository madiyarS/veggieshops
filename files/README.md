## 🎯 ОСНОВНЫЕ МОДУЛИ СИСТЕМЫ

### 1. Клиентская часть (для покупателей)
- Каталог товаров с фильтрацией
- Корзина товаров
- Оформление заказа (4 этапа)
- Выбор доставки (Regular/Express)
- Выбор района доставки
- Проверка радиуса доставки
- Выбор времени доставки (2-часовые слоты)
- Платежи (Kaspi.kz, Halyk Bank, наличные)
- Отслеживание заказа по номеру
- WhatsApp уведомления

### 2. Админ-панель (для управления)
- Управление магазинами (неограниченное количество)
- Управление товарами (50-70+ позиций)
- Управление заказами (полный цикл статусов)
- Управление районами доставки
- Управление временными окнами доставки
- Назначение курьеров
- Интеграция платежей
- Аналитика и отчеты
- Управление пользователями

### 3. Система доставки (самое важное!)
- **Радиус доставки:** настраивается в админ-панели (например, 3км)
- **Районы:** администратор создает районы внутри радиуса
- **Проверка расстояния:** рассчитывается по формуле Хаверсина
- **Разная стоимость:** Regular доставка vs Express доставка
- **Временные окна:** 2-часовые слоты (09:00-11:00, 11:00-13:00 и т.д.)
- **Управление курьерами:** распределение заказов
